data.table joins: is X[Y] or Y[X]?

This is intended to be a very short post, an often-revisited reminder of the syntax for joins in data.table.

When doing joins in data.table I often have to open a new R session and run the next code:

a <- data.table(x = 1:3, y = 2:4)
b <- data.table(x = 2:5, z = 3:6)
a[b, on = .(x)]
##    x  y z
## 1: 2  3 3
## 2: 3  4 4
## 3: 4 NA 5
## 4: 5 NA 6

in order to realize that the look-up table was a and not b. It seems I tend to read it as “take a and look up its x column in b”, which is obviously wrong.

It should be read as “show me the table a having b”.

So from now on my mnemonic will be “What’s going on in the inside shows on the outside”1.
